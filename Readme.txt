Общее описание программы.

Программа построена на основе фреймворка PlayFramework 1.5. и реализует web-сайт о ставках на футбольные матчи.
(Версия 1.2.7.2 была не совместима с используемой версией jdk, поэтому быля взята 1.5).
В приложении реализовано две проли: администратор и пользователь. Роли добавляются в БД в тестах.
	На главной странице выводится список матчей. Пользователь может выбрать любой матч и попытаться угадать результаты матча.
Если пользователь угадал победителя, то выводится сообщение "Вы угадали победителя". Если пользователь
угадает счет, то выводится сообщение "Вы угадали счет.". Если пользователь ничего не угадает, то выведется 
сообщение "Вы ничего не угадали.".
	На главной странице есть ссылка в админку, где можно создать команды и матчи. Только пользователь с правами
адиминистратора может выполнить эти действия.

Ограничения:
	Интерфейс сделан в упрощенном биде, без применения стилей.
	Роль пользователя выполнена в качестве булевского флага в таблице пользователй. Правильней сделать роли отдельной 
сущьностью. 
	Для упрощения, счет матча сделал в виде отдельных полей. В дальнейшем можно сделать одним полем с выполнением 
анализа счета.
